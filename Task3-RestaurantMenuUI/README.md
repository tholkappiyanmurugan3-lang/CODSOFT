# SpiceHaven - Restaurant Menu UI

SpiceHaven is a modern restaurant landing page and menu UI designed using Figma.

## Features
- Modern Navigation Bar
- Hero Banner Section
- Food Categories
- Popular Dishes Showcase
- Special Offer Banner
- Professional Footer

## Tools Used
- Figma
- UI/UX Design Principles
- Visual Design
